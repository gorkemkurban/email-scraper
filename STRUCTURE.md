# Project Structure

## Core Python Files
- **gui.py**: Graphical user interface (Tkinter-based)
- **main.py**: Command-line interface
- **config.py**: Configuration settings and constants
- **excel_handler.py**: Excel file I/O operations
- **email_extractor.py**: Email pattern detection and validation
- **web_scraper.py**: Website crawling and email extraction
- **page_detector.py**: Multi-language contact page detection

## Configuration Files
- **requirements.txt**: Python dependencies
- **EmailScraper.spec**: PyInstaller build configuration
- **.gitignore**: Git ignore patterns
- **LICENSE**: MIT License

## Utility Scripts
- **run.bat**: Windows launcher with dependency installation
- **build.bat**: PyInstaller build script

## Documentation
- **README.md**: Main project documentation

## Key Features
- 8-layer email detection system
- 11-language support for contact page detection
- Smart filtering (removes personal/system emails)
- Multi-sheet Excel support with column normalization
- Real-time progress tracking and logging
- Standalone executable generation

50-60% typical success rate (varies by website structure)
